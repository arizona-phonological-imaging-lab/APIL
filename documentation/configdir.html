Configure Directories

This script arranges training data into the directory structure that TrainNetwork.py expects.

Usage

  • run configdir.py by typing python configdir.py
  • Select the train directory created by ImageDiversity
  • Running this script produces the directories Subject1/, traces/ and Subject1/IMAGES/, as well as TongueContours.csv.
  • ROI_config.txt is moved into Subject1/.
