Analysis Tools

There are two manuals for trace analysis tools that are included with Autotrace.

  • Linguaview - a visualizing and analysis tool that includes a linguagram waveform, and spectrogram

  • Analysis gui frontend for creating neutral subtraction linguagrams, or sequential linguagrams for a number of trace files
